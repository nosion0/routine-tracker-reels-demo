# 릴스 데모 화면 검증 기록

검증 일시: 2026-05-07

로컬 미리보기 서버 `http://localhost:4173/`에서 앱을 열고 **월간** 탭으로 이동했다. 월간 화면 상단에 **릴스 촬영 모드** 히어로 영역과 **데모 버튼**이 정상 표시되었으며, 기존 월간 습관표 제목, 원형 차트, 꺾은선 차트, 체크박스 테이블도 함께 렌더링되었다.

데모 버튼을 클릭하자 버튼 문구가 **데모 다시보기**로 바뀌고 **실제 데이터 보기** 버튼이 나타났으며, 진행률 표시가 100%까지 동작했다. 원형 차트 색상이 보라색 계열에서 주황색 계열로 전환되었고, 꺾은선 차트도 데모 체크 상태를 반영해 선과 점이 변경되었다. 월간 체크박스에는 주황·분홍·보라 그라디언트 강조 효과가 적용되어 자동 채움 연출이 시각적으로 확인되었다.

브라우저 미리보기 검증 스크린샷:

- 초기 월간 화면: `/home/ubuntu/screenshots/localhost_2026-05-07_09-11-30_9436.webp`
- 데모 실행 후 화면: `/home/ubuntu/screenshots/localhost_2026-05-07_09-11-47_9481.webp`

빌드 명령 `pnpm build`는 TypeScript, Vite, Electron main/preload 번들 모두 성공했다.

추가로 데모 완료 상태에서 체크박스 하나를 클릭해 보니 **데모 다시보기/실제 데이터 보기** 상태가 해제되고 버튼이 다시 **데모 버튼**으로 돌아왔다. 원형 차트와 꺾은선 차트도 실제 체크 상태 기반의 보라색 데이터 표시로 복귀해, 사용자가 데모 중 체크박스를 누르면 실제 데이터 모드로 돌아가는 요구사항을 충족함을 확인했다.

- 실제 데이터 복귀 확인 화면: `/home/ubuntu/screenshots/localhost_2026-05-07_09-12-25_9784.webp`
